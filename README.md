# Shooting Game — Classic Browser-Based Shooter by Deividas Strole

Shooting Game is a fun, lightweight shooting game built entirely with **vanilla JavaScript**, **HTML5 Canvas**, and **CSS**. Created by Deividas Strole, this project brings fast-paced arcade-style shooting action directly to your browser—no frameworks, no dependencies, just pure web tech.

Control your ship (or character), dodge enemies, and shoot incoming threats while racking up points. Perfect for quick gaming sessions or as a learning example of canvas-based game development.

This repository, developed by Deividas Strole, is part of his portfolio showcasing interactive JavaScript games and real-time rendering.

🚀 Overview

Survive waves of enemies by shooting them down before they reach you. The game features smooth movement, bullet physics, collision detection, scoring, and a game-over screen — all implemented from scratch using HTML5 Canvas.

Built by Deividas Strole to demonstrate core game programming concepts like game loops, input handling, and object management in JavaScript

✨ Key Features

- Smooth player movement and shooting mechanics
- Enemy spawning and movement patterns
- Bullet-enemy collision detection
- Real-time score tracking
- Game over and restart functionality
- Responsive canvas that adapts to window size
- Single-file or minimal structure for easy understanding
- No external libraries — pure vanilla JS + Canvas

🛠️ Built With

| Technology    | Purpose                              |
|---------------|--------------------------------------|
| JavaScript    | Game logic, input, rendering loop    |
| HTML5 Canvas  | 2D graphics and animation            |
| HTML & CSS    | Page structure and basic styling     |

- Building a complete game using only browser-native technologies

- Handling keyboard/mouse input for controls

- Creating engaging, playable web experiences

Great for JavaScript learners, game dev enthusiasts, or anyone wanting a fun retro-style shooter.
